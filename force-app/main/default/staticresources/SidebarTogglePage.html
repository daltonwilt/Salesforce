Sidebar Toggle Demo

Welcome

This page demonstrates floating icons in the lower right corner. Click on any icon to toggle the corresponding sidebar panel.

Each sidebar is 30% of the screen width and slides in from the right side.

  • Settings Icon - Toggle Settings Sidebar
  • Notification Icon - Toggle Notifications Sidebar
  • Info Icon - Toggle Information Sidebar

About This Page

This demonstration page showcases the use of SLDS (Salesforce Lightning Design System) to create a modern interface with floating action buttons and toggleable sidebars.

Features

  • Floating icons with hover effects
  • 30% width right-aligned sidebars
  • Smooth slide-in/out animations
  • SLDS styling and components
  • Responsive design
